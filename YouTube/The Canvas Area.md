## Print Frames

设置默认打印框架（Setting a default frame）
如果你大多数 TCP 都使用同一个框架，可以把它设为默认。
步骤：
- 将框架保存为 .tcpf
- 打开 File → Print → Print Options
- 点击 “Edit default print frame settings”
- 选择你的 .tcpf 文件
- 勾选 “Use as default print frame”
- 保存设置
之后所有没有指定框架的图都会自动使用这个默认框架。
